# Security Policy

## Reporting a vulnerability
- Please report security issues privately via email: security@gitsummary.dev
- Include reproduction steps, affected versions/commits, and any possible impact.
- We aim to acknowledge reports within 5 business days.

## Supported versions
- The latest main branch and the most recent tagged release are supported.

## Public disclosure
- We will coordinate disclosure timing with reporters. Please do not file public issues for potential vulnerabilities until we respond.
